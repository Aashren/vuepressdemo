Assembly




Task 1

1. Connect Battery holder with EPS, using M3 L6 Screw and M3 Nut.

2. Put Battery inside the Battery holder (Checking +ve and -ve Battery Termnial).

3. Connect Deployment switch to connector as shown in figure.

4. Connect Solar cell to connector as shown in figure.

5. Connect Battery Connector as shown in figure.

6. Face Solar cell towards any light source and check if the LED in EPS glows.

7. Connect 2 Pin header on current named male header on EPS PCB.

8. Connect Type-B cable as shown in figure.

9. Check if the LED in EPS glows or not.

Task 2

1. Connect 40 Pin Header on OBC header Pin.

2. Connect 10mm and 15mm standoff as in figure.

3. Connect another set of 15mm standoff as in figure.

4. Place EPS board as shown uin figure.

5. Connect 15mm standoff as in fifure.

6. Place MSN board as shown in figure.

7. Connect 10mm Spacers as shown in figure.

8. Connect 10mm standoff to spacers as shown in figure (This Step is optional, follow only if extra space between boards is required)

Task 3

1. Place Electronics inside a frame.

2. Connect assembled boards together with Structure using M3 L12 screw.(Repeat same step for next frame on -Y side and screw using M3 L12 screw)

3. Connect assembled boards together with Structure using M3 L12 screw from bottom too.(Repeat same step for next frame on -Y side and screw using M3 L12 screw)

4. Place double sided tape on -Y acrylic.

5. Attach solar cell on -Y acrylic as shown in figure.

6. Use M3 L6 screw to attach -Y acrylic on -Y frame as shown in figure.

7. Use M3 L6 screw to attach +Y acrylic on +Y frame as shown in figure.

8. Use M3 L6 screw to attach +X acrylic on +X frame as shown in figure.

9. Attach Deployment switch on -X acrylic using M2 L12 screw and M2 Nut.

10. Use M3 L6 screw to attach -X Acrylic on -X frame as shown in figure.
